# 🚗 Car Black Box System (Embedded C)

## 📌 Project Overview

The **Car Black Box System** is an embedded system project designed to record and monitor critical vehicle data such as speed, events, and time.

This system helps in **accident analysis** and **driver behavior monitoring**, similar to an aircraft black box.

---

## 🚀 Features

✨ Real-time vehicle data monitoring  
✨ Event logging (acceleration, braking, collisions)  
✨ Speed tracking using sensors  
✨ Time-stamped data recording  
✨ Display data on LCD  
✨ Store logs in memory (EEPROM)  
✨ Serial communication (UART) for data transfer  

---

## 🛠️ Technologies Used

- **Programming Language:** C  
- **Platform:** Microcontroller (Embedded System)  
- **Concepts:**
  - ADC (Analog to Digital Conversion)
  - Timers
  - Interrupts
  - UART Communication

## ⚙️ Working Principle

1. Sensors collect vehicle data (speed/events)  
2. Microcontroller processes input signals  
3. Data is displayed on LCD  
4. Events are stored in EEPROM  
5. Logs can be transmitted using UART
